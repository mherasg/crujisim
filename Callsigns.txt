# Some missmatch has been found during this databse "manual" integration, so the following criteria has been applied:
#
#   If the CALLSIGN is the same in both databases the callsign appears as it is. Example: IBERIA
#   If the CALLSIGN only appears in one of the databases the callsign appears as it is. Example: IBERIA
#   If the CALLSIGN does not appear in neither database, the company name according to http://www.airframes.org/airlines/ is used and is written in brackets. Example [OAG Worldwide]
#   If the CALLSIGNS in both databases are different SENASA's CALLSIGN appears first and http://www.airframes.org/airlines/'s in parentheses. POLLOT (LOT)
#   If the CALLSIGN appears in SENASA's and does not in http://www.airframes.org/airlines/'s. SENASA's apppears as main and also the company's name in parentheses. Example  EXCELLENCE (Hapag-Lloyd Express)
#   If no callsign has been found [] appears
#Exceptions: RAF has been kept as Royal Air Force but actually its Farnas Aviation Services and has no CALLSIGN.
