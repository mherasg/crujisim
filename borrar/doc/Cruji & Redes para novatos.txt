-Tutorial Cruji&Redes para novatos como yo-


1. Puertos

- Router de Telefonica: hacer 192.168.1.1 (puerto del router), clave y pass suelen ser 1234 por defecto, ir a NAT para abrir puertos para la IP física del/de los pc(s)*, creando una regla para cada combinación puerto-pc.

-Si firewall tiene bloqueado un puerto crear una regla específica en el firewall (en opciones avanzadas).

- Puerto por defecto del cruji es 20123,

para crujir varios con un router dentro de casa basta con poner la IP física del pc* al que te quieres conectar,
 
para crujir varios por internet simultáneamente abrir puerto diferente para cada usuario en router y firewall y editar con bloc de notas el archivo cruji.ini** para añadir la línea:
server_port = 20124 (p.e. ,siendo 20124 el puerto que has elegido).
Cuando creas pasada el ATC remoto se conectará a ti poniendo
83.61.32.199**:20124 (p.e.)


*para verla inicio-ejecutar-"cmd", en la consola que aparece "ipconfig" y aparece:
Dirección IP.........sosudicha IP física del pc
Máscara de subred....(sin importancia)
Puerta...............la IP por la que te comunicas con tu router

**cruji.ini es el del icono del texto+engranaje :).

**para conocer tu IP de internet http://www.myip.dk/.


-Si el cruji te da problemas prueba a borrar la carpeta y hacer un update en una carpeta nueva vacía.
